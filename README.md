# Media-Edukasi-Refleksi-Diri-Anti-Bullying
SpeakUp! adalah media edukasi dan refleksi diri berbasis web yang dirancang untuk meningkatkan kesadaran terhadap isu bullying. Melalui fitur pembelajaran dan form refleksi, pengguna diajak memahami bentuk-bentuk bullying, mengekspresikan perasaan, serta belajar menjadi pribadi yang empatik dan berani bersuara.
